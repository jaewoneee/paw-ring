빌드 검증 후 Git push를 수행해줘.

## 프로세스

1. `git status`로 현재 브랜치와 상태를 확인한다.
2. **스테이징된 변경사항이 있으면**: /commit 프로세스를 자동 실행하여 커밋한 뒤 계속 진행한다.
3. **스테이징된 변경사항도 없고 push할 커밋도 없으면**: "push할 내용이 없습니다"라고 알려주고 종료한다.
4. `npx expo export` 명령으로 빌드를 실행한다.
5. **빌드 실패 시**: 에러 내용을 사용자에게 보여주고 push를 중단한다. 절대 push하지 않는다.
6. **빌드 성공 시**: `git push`를 실행한다.
7. push 결과를 사용자에게 알려준다.
