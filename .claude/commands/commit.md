Git 스테이징된 파일을 확인하고 커밋해줘.

## 프로세스

1. `git diff --cached --stat`으로 스테이징된 파일 목록을 확인한다.
2. 스테이징된 파일이 없으면 "스테이징된 파일이 없습니다"라고 알려주고 종료한다.
3. `git diff --cached`로 변경 내용을 분석한다.
4. `git log --oneline -5`로 최근 커밋 메시지 스타일을 참고한다.
5. 변경 내용을 바탕으로 적절한 커밋 메시지를 작성한다.
   - Conventional Commits 형식 사용 (feat, fix, refactor, chore, docs 등)
   - 한국어로 작성
   - 간결하고 명확하게
6. 작성한 커밋 메시지를 보여주고 바로 커밋을 실행한다. (별도 확인 불필요)
