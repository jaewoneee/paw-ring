기획 문서를 기반으로 다음에 구현할 기능을 추천하고, 승인 시 구현해줘.

## 프로세스

1. `.claude/product/feature/README.md`를 읽어 전체 기능 목록과 상태를 파악한다.
2. 우선순위 로드맵(Phase 1 → Phase 2 → Phase 3) 순서를 따른다.
3. 현재 Phase에서 아직 미착수(⬜)이거나 진행중(🔧)인 기능 중, 다음 규칙으로 1개를 추천한다:
   - 같은 Phase 내에서는 번호가 빠른 것을 우선한다.
   - 진행중(🔧) 항목이 있으면 미착수(⬜)보다 우선한다 (마무리 우선).
   - 의존성이 있는 기능은 선행 기능이 완료(✅)된 후에만 추천한다.
4. 추천 시 다음 정보를 포함한다:
   - 기능 번호와 이름
   - 해당 Phase 정보
   - 상세 기획 문서가 있으면 해당 문서의 핵심 요구사항 요약
   - 예상 작업 범위 (어떤 파일/화면을 추가·수정해야 하는지)
5. 사용자가 승인하면 (ok, 좋아, ㅇㅇ, 고고, 등) 해당 기능을 구현한다.
6. 구현 완료 후 `.claude/product/feature/README.md`와 관련 상세 기획 문서의 상태를 업데이트한다.
