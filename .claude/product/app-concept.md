# Paw Ring - 반려동물 스케줄러 앱 컨셉

## 개요

반려동물에 특화된 스케줄러 앱. 반려동물 단위로 캘린더를 관리하고, 반복 스케줄 생성 및 푸시 알림을 제공한다.

---

## 핵심 기능

### 1. 반복 스케줄링

- 유저가 설정한 기간(주기)에 따라 스케줄이 자동 반복 생성
- 예: 매일 / 매주 / 격주 / 매월 등

### 2. 푸시 알림

- 유저가 설정한 타이밍에 맞춰 푸시 알림 전송
- 스케줄 시작 전 알림 시점 커스터마이징 가능

### 3. 캘린더 공유

- 반려동물 단위의 캘린더를 다른 유저와 공유
- 예: A가 "테오"(강아지) 캘린더를 생성 → B에게 공유 → B도 함께 열람 가능
- 공동 돌봄 시나리오 지원

### 4. 멀티 캘린더

- 유저는 여러 개의 캘린더(스케줄러)를 생성 가능
- 캘린더는 **반려동물 단위**로 동작 (1 반려동물 = 1 캘린더)

---

## 구조

```
유저
 ├── 반려동물 A (캘린더 A)
 │    ├── 스케줄 1 (반복: 매일)
 │    ├── 스케줄 2 (반복: 매주)
 │    └── ...
 ├── 반려동물 B (캘린더 B)
 │    └── ...
 └── 공유받은 캘린더
      └── 반려동물 C (from 다른 유저)
```
