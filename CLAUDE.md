# Project: PawRing

React Native Expo 반려동물 앱. theo-kit-native를 UI 컴포넌트 라이브러리로 사용.

## Tech Stack
- Expo SDK 55 + Expo Router (file-based routing)
- NativeWind v4 + Tailwind CSS v3
- theo-kit-native / theo-kit-core (로컬 file: 참조)
- yarn

## Commands

### /commit
변경사항을 확인하고 적절한 커밋 메시지로 커밋한다. 푸시는 하지 않는다.

### /push
변경사항을 확인하고 적절한 커밋 메시지로 커밋한 뒤 `git push -u origin main`으로 푸시한다.
